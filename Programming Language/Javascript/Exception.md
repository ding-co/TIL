## [구문 오류와 예외]

<br/>

### _구문 오류_

- 프로그램 실행 전에 발생하는 오류
- SyntaxError <br/>
  ex) 괄호 짝, 문자열 열고 닫기, ...

<br/>

### _예외 (런타임 오류)_

- 프로그램 실행 중에 발생하는 오류 <br/>
  (오류 발생 하지 않은 부분은 정상 실행)
- Exception <br/>
  ex) 식별자 이름 잘못 적기, 네트워크 통신 오류로 데이터 받지 못함, ...

#

### [Note]

- 자바스크립트: 동적 언어; 구문오류 제외한 모든 오류 => 예외
